# bender-zyn-scanner
This will open your camera. And you put the zyn QR code in front of the camera. It will automatically enter it into the zyn website for you to get your rewards.


First you have to make sure you have chrome installed. Then you have to run runchrome.ps1 to get the chrome instance with remote debugging running. Then you can run python3 main.py to run the program and open the camera window.
